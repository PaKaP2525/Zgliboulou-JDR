Caractéristiques :
STR/FOR : Force
DEX : Dextérité
CON/CONS : Constitution
INT : Intelligence
WIS/SAG : Sagesse
CHA/CHAR : Charisme

DD : Difficulté du Dé (aka combien il faut faire pour passer le check)
CC : Crowd Control, l'ensemble des effets débilitants (confusion, aveuglement, etc)
Résistance : les dégâts reçus de ce type sont diminués de moitié (minimum 1)
Vulnérabilité : les dégâts reçus de ce type sont multipliés par 2
