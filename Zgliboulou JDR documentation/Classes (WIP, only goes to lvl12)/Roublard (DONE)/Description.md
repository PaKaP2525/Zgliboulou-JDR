Le roublard est une classe martiale, qui se distingue des autres classes martiales par son manque de confrontation directe ; malin et discret, il se faufile derrière les ennemis pour infliger des dégâts massifs.

Caractéristique principale : Dextérité
Stat secondaire : Charisme
Dé de PV : 1d8
Dé de PE : 1d4

Il faut avoir +10 en Dextérité pour multiclasser Roublard.
