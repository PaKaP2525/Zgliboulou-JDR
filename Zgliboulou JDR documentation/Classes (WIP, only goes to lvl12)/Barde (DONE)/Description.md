Le barde est une classe très particulière, qui inflige peut de dégâts directs ; elle se spécialise dans les buffs/débuffs et les interactions sociales

Caractéristique principale : Charisme
Caractéristique secondaire : N'importe laquelle
Dé de PV : 1d4
Dé de PE : 1d4

Il faut avoir +10 en Charisme pour multiclasser Barde.
